AI 사용량 위젯

Claude · Codex 5시간/주간 사용량을 항상 화면에

처음 시작하기

  1. 트레이 아이콘 우클릭
  2. "Claude 로그인 창 열기" 또는 "Codex 로그인 창 열기" 클릭
  3. 평소처럼 로그인 (Google / 이메일 / SSO 등)
  4. 로그인 완료되면 창이 자동으로 닫히고 실제 사용량이 표시돼요

트레이 메뉴 살펴보기

  • 위젯 카드로 보기 / 트레이 아이콘으로만 보기 — 표시 방식 선택
  • 항상 위로 고정 — 켜면 다른 창 위에 항상 떠 있음
  • 위젯 투명도 / 색상 테마 — 취향대로 꾸미기
  • Claude 표시 / Codex 표시 — 서비스별로 켜고 끄기
  • 지금 새로고침 — 즉시 다시 불러오기

위젯 카드는 아무 곳이나 드래그해서 원하는 위치로 옮길 수 있어요. 로그인 세션은 앱에 저장되어 다시 로그인할 필요가 없어요.
